custom-notebooks
================

Code for generating my custom notebooks. See [this post][1] for details.

Custom Notebook Setup
---------------------

1. Install dependencies:
    - Python 2.7
    - Install [python-qrcode][2]
2. Build the notebook with the Makefile.
